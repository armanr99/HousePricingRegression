Linear Regression¶

Linear regression is a method for modeling the relationship between one or more independent variables and a dependent variable. In simple model we assume that $\hat{y}$ is a linear function of the input variable as shown below:

$$ \hat{y} = w.x + b $$

w and b are called co-efficients and x is input. We'll try to guess co-efficients to have a better y function for predicting and minimizing the error function. In other words, we'll try to find the line which the vertical distance between line and data represented as point which is the difference between predicted and real data is minimized. There are several methods to choose co-efficients. Two methods are implemented here: Normal Equation and Gradient Descent. We'll use TotalBsmtSF as input since It has a better linear relationship with price based on previous part diagrams.

Error function¶

We use RMSE method as error function to determine difference between real price and predicted price which is shown below:

$$ L_{RMSE} = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(\hat{y}_{i} - y_{i})^2} $$

The code below shows implementation of RMSE error function:

In [18]:
def get_rmse(y_real, y_hat):
    N = y_real.shape[0]
    return np.sqrt(np.sum(np.power(y_hat - y_real, 2)) / N)

Normal Equation¶

In this method, we'll assume x and co-efficients as matrixes and try to solve the problem directly using a system of linear equations. In simple model which has only one independent variable we have the below formulas:

$$ \hat{y} = B.X = \begin{bmatrix} b & w \end{bmatrix} . \begin{bmatrix} 1 \\ x \end{bmatrix} $$

We'll try to solve the equation directly:

$$ \hat{y} = B.X $$
$$ X^T.\hat{y} = B.X.X^T $$
$$ B = (X.X^T)^{-1}.X^T.\hat{y} $$

Below code shows the implementation of this method:

In [19]:
from numpy.linalg import inv

def normal_equation(data, dependent, independent):
    y_hat = np.matrix(data[independent].values).T

    X_arr = data[dependent].values
    X_size = X_arr.shape[0]
    X = np.matrix([np.ones(X_size), X_arr]).T

    B = inv(X.T.dot(X)).dot(X.T).dot(y_hat)
    w = B[1, 0]
    b = B[0, 0]
    
    return (w, b)

Gradient Descent¶

Gradient Descent is an algorithm which minimizes functions by taking steps in the negative direction of the function gradient. We start by initial set of co-efficients and iteratively move forward a set of parameters to minimize the error function. First we compute the gradient of error function:

$$ L_{RMSE} = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(y_{i} - \hat{y}_{i})^2} = \sqrt{\frac{1}{N}\sum_{i=1}^{N}({y}_{i} - w.x_{i} + b)^2} $$
$$ \frac{\partial{L_{RMSE}}}{\partial w} = \frac{1}{N}. \frac{\sum_{i=1}^{N} -x_i({y}_i - (w.x_i + b)) } {\sqrt{\frac{1}{N}\sum_{i=1}^{N}(y_{i} - w.x_{i} + b)^2}} $$
$$ \frac{\partial{L_{RMSE}}}{\partial b} = \frac{1}{N}. \frac{\sum_{i=1}^{N} -(y_i - (w.x_i + b)) } {\sqrt{\frac{1}{N}\sum_{i=1}^{N}(y_{i} - w.x_{i} + b)^2}} $$

We'll use gradient of error function to change values of w and b in direction of getting less error. We also use learning rate as lRate which controls how much the value of w changes on each iteration. values of w and b change as follows:

$$ w = w - lRate . \frac{\partial{L_{RMSE}}}{\partial{w}} $$
$$ b = b - lRate . \frac{\partial{L_{RMSE}}}{\partial{b}} $$

Code below shows the implementation of Gradient Descent:

In [38]:
def gradient_descent(data, dependent, independent):

    lRate = 0.0001
    lTimes = 1000
    w = 0
    b = 0

    X = data[dependent].values
    y = data[independent].values
    y_hat = 0

    N = X.shape[0]

    for i in range(lTimes):
        y_hat = w * X + b
        denominator = np.sqrt((1 / N) * sum((y - y_hat) ** 2))
        d_w = ((-1 / N) * sum(X * (y - y_hat))) / denominator
        d_b = ((-1 / N) * sum(y - y_hat)) / denominator
        w -= (lRate * d_w)
        b -= (lRate * d_b)
        
    return (w, b)

K-nearest Neighbours¶

K-nearest Neighbours is a non-parametric method to predict numerical targets. in this algorithm we'll use k nearest neighbours based on distance and report average of k neighbours's target values as goal. In this project we'll use Euclidean distance as distance function which is shown below:

$$ EuclideanDistance = \sqrt{\sum_{i=1}^{k}(x_i - y_i)^2} $$

Standardize Distances¶

We standardize our training set to balance measurement scales or if we have categorical values, we'll try to standardize them to numerical values. We'll use the below formula to standardize our training set:

$$ X_s = \frac{X - Min}{Max - Min} $$

The code below shows the standardize implementation for our dataset:

In [27]:
def standardize(data):
    Min = data.min()
    Max = data.max()
    return (data - Min) / (Max - Min)

Code below shows implemenation of KNN algorithm:

def get_knn_price(single_data):
    
    k = 10
    
    clean_data = delete_columns(data, ['Id', 'SalePrice'])
    appended_data = clean_data.append(single_data, ignore_index=True)
    
    clean_data = standardize(appended_data)  
    single_data = clean_data.iloc[-1]
    clean_data = clean_data[:-1]
    
    clean_array = clean_data.to_numpy()
    single_array = single_data.to_numpy()
    
    e_distance = np.sqrt(np.power(clean_array - single_array, 2).sum(axis = 1))
    
    result_data = pd.DataFrame({"EDistance": e_distance, "SalePrice": data["SalePrice"]})
        
    k_smallest = result_data.nsmallest(k, "EDistance")
    k_average = k_smallest["SalePrice"].mean()
    
    return k_average
